# Roadmap

## Near Term

- Add document delete and re-index flows
- Add support for DOCX and plain text uploads
- Improve quiz grading beyond substring matching
- Add section-level extraction and richer chunk metadata

## Next Phase

- Replace keyword retrieval with persisted embeddings and vector search
- Add audit logs for document access and answer generation
- Add team or department-based document visibility
- Add admin reporting for completion by cohort

## Production Hardening

- Add migrations and production database support
- Add test coverage for auth, ingestion, retrieval, and quiz flows
- Add background processing for large document ingestion
- Add SSO support for enterprise environments
- Add rate limits and upload validation safeguards
