# About

Backend and UI for the Study Buddy project.

### Requirements

* [Java 19](https://jdk.java.net/19/)
* [Node.js](https://nodejs.org/en/)

### Current Dependencies

* Spring Boot
* Spring Boot Dev Tools
* Spring Boot JPA
* Spring Boot Web
* Lombok
* Bootstrap ([Examples](https://ng-bootstrap.github.io/#/components/accordion/examples))
